# BookwormPRO (Sale Edition)

Pre-compiled distribution with 6 Cython-compiled modules and 135 encrypted skills.

## Requirements

- Python 3.12+
- Windows or Linux

## Install

```bash
git clone https://github.com/huakoh/bookwormpro-sale.git
cd bookwormpro-sale
pip install -e ".[all]"
```

## License Activation

```bash
export BOOKWORMPRO_LICENSE_KEY="your-license-key"
```
