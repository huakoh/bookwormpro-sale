# 中国法务审查专家技能

> **Output Style**: 本技能使用内联输出规范

## 概述

本技能为Claude提供专业的中国法务审查能力，涵盖合同审查、合规评估、风险分析等核心法务工作场景。

**重要声明**：本技能提供的分析和建议仅供参考，不构成正式法律意见。涉及重大法律事项，建议咨询持证执业律师。

## 核心审查流程

法务审查遵循以下标准流程：

1. **文档识别** → 判断文档类型与适用法律领域
2. **条款提取** → 识别关键条款与核心义务
3. **合规检查** → 对照适用法律法规进行合规性分析
4. **风险评估** → 识别法律风险并评估风险等级
5. **意见输出** → 生成结构化审查意见与修改建议

## 审查领域与适用法规

### 1. 合同审查

**适用法规**：《民法典》合同编、相关司法解释

**审查要点**：主体资格、合同效力、权利义务对等性、违约责任、争议解决条款、格式条款效力

**详细指南**：参见 `references/contract-review.md`

### 2. 劳动用工审查

**适用法规**：《劳动法》《劳动合同法》《社会保险法》

**审查要点**：劳动合同必备条款、试用期合规、竞业限制、解除终止条件、经济补偿

**详细指南**：参见 `references/labor-review.md`

### 3. 数据合规审查

**适用法规**：《个人信息保护法》《数据安全法》《网络安全法》

**审查要点**：合法性基础、知情同意、敏感信息处理、跨境传输、安全保护义务

**详细指南**：参见 `references/data-compliance.md`

### 4. 知识产权审查

**适用法规**：《著作权法》《专利法》《商标法》《反不正当竞争法》

**审查要点**：权利归属、许可范围、侵权风险、商业秘密保护

**详细指南**：参见 `references/ip-review.md`

### 5. 公司治理审查

**适用法规**：《公司法》《证券法》

**审查要点**：章程合规性、决议程序、关联交易、对外担保、信息披露

**详细指南**：参见 `references/corporate-review.md`

## 风险等级评定标准

| 等级 | 定义 | 建议 |
|------|------|------|
| **高风险** | 可能导致合同无效、重大经济损失或行政/刑事责任 | 必须修改后方可签署 |
| **中风险** | 存在较大争议可能或一定经济损失风险 | 强烈建议修改 |
| **低风险** | 存在轻微瑕疵但不影响主要权利义务 | 建议优化完善 |
| **提示** | 非风险事项，但需关注或知悉 | 供决策参考 |

## 使用指引

### 合同审查任务

1. 判断合同类型（买卖、服务、劳动、投资等）
2. 加载对应 `references/` 参考文件
3. 逐条审查关键条款
4. 按风险等级分类问题
5. 输出结构化审查意见书

### 合规咨询任务

1. 明确咨询的业务场景
2. 识别适用的法律法规
3. 分析合规要点与义务
4. 提供合规建议与整改方案

## 重要注意事项

1. **时效性**：法律法规持续更新，重要事项建议查阅最新法规
2. **地域性**：部分规定存在地方差异，需关注适用地区
3. **专业性**：复杂事项建议咨询专业律师
4. **免责声明**：始终包含"不构成正式法律意见"的声明
