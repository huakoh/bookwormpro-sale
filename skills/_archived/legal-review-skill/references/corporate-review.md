# 公司治理与商事法律审查指南

## 适用法规

- 《中华人民共和国公司法》（2023年修订，2024年7月1日施行）
- 《中华人民共和国证券法》（2019年修订）
- 《上市公司章程指引》
- 《上市公司股东大会规则》
- 《上市公司治理准则》

## 公司章程审查要点

### 必载事项

**有限责任公司章程**（《公司法》第46条）：
1. 公司名称和住所
2. 公司经营范围
3. 公司注册资本
4. 股东的姓名或者名称
5. 股东的出资额、出资方式和出资时间
6. 公司的机构及其产生办法、职权、议事规则
7. 公司法定代表人的产生办法
8. 股东会认为需要规定的其他事项

**股份有限公司章程**（《公司法》第93条）另需载明：
- 公司股份总数、每股金额
- 发起人的姓名或名称、认购的股份数
- 董事会、监事会的组成、任期和议事规则

### 高风险条款类型

| 条款类型 | 常见问题 | 风险 |
|----------|----------|------|
| 股东权利限制 | 不合理限制股东知情权、分红权 | 条款可能被认定无效 |
| 表决权安排 | 与出资比例脱钩无合法依据 | 需依法约定 |
| 股权转让限制 | 过度限制股权转让自由 | 可能被认定无效 |
| 公司担保条款 | 未规定决策程序 | 导致担保行为效力争议 |

## 股东会/董事会决议审查

### 决议效力类型

**《公司法》第25-27条**：

| 效力类型 | 情形 | 后果 |
|----------|------|------|
| 不成立 | 未召开会议、未表决、未达法定人数或比例 | 决议自始不存在 |
| 可撤销 | 召集程序、表决方式违法或章程 | 起诉后法院可撤销 |
| 无效 | 决议内容违反法律、行政法规 | 自始无效 |

### 股东会决议审查清单

**召集程序**：
- [ ] 有权召集主体（董事会/监事会/符合条件的股东）
- [ ] 通知时间（有限公司15日/股份公司20日，章程另有规定除外）
- [ ] 通知方式符合章程规定
- [ ] 通知内容包含会议时间、地点、议题

**表决程序**：
- [ ] 出席人数/股份比例符合要求
- [ ] 表决比例符合法律和章程规定
- [ ] 特别决议事项（修改章程、增减注册资本、合并分立解散等）经2/3以上表决权通过
- [ ] 关联股东回避表决（如适用）

## 对外担保审查

### 法律依据

**《公司法》第15条**：公司向其他企业投资或者为他人提供担保，依照公司章程的规定，由董事会或者股东会决议。

**《民法典担保制度司法解释》第7-12条**：明确了公司担保的效力规则

### 对外担保决策权限

| 担保对象 | 决策机构 | 法律依据 |
|----------|----------|----------|
| 为股东或实际控制人担保 | 必须股东会决议 | 《公司法》第15条 |
| 为非关联方担保 | 董事会或股东会（依章程） | 《公司法》第15条 |
| 担保总额或单项超限 | 通常需股东会决议 | 依章程规定 |

### 担保决议审查要点

1. **决议机关正确**：是否符合章程规定的决策权限
2. **关联回避**：为股东担保时，该股东不得参与表决
3. **表决比例**：是否达到章程规定的表决比例
4. **决议真实性**：签字、盖章真实有效

### 善意相对人保护

**《民法典担保制度司法解释》第7条**：
相对人已对决议进行合理审查的，应认定其构成善意。合理审查包括：
- 决议的表决程序符合法律规定
- 签字人员符合章程载明的人员
- 决议签章真实

## 关联交易审查

### 关联方界定

**《公司法》相关规定**及会计准则：

关联方通常包括：
- 控股股东、实际控制人
- 董事、监事、高级管理人员
- 前述人员的近亲属
- 前述人员直接或间接控制的企业
- 其他可能导致利益转移的关系方

### 关联交易审批程序

| 关联方 | 回避表决 | 审批机关 |
|--------|----------|----------|
| 董事关联交易 | 关联董事回避 | 董事会/股东会 |
| 股东关联交易 | 关联股东回避 | 股东会 |
| 上市公司关联交易 | 关联方回避 | 独立董事发表意见 |

### 上市公司特别要求

**《上市公司治理准则》**：
1. 关联交易应当签订书面协议
2. 关联交易定价应当公允
3. 重大关联交易须经独立董事事前认可
4. 须履行信息披露义务

## 股权转让审查

### 有限责任公司股权转让

**《公司法》第84条**：
- 对内转让：自由转让
- 对外转让：书面通知其他股东；其他股东过半数同意
- 优先购买权：同等条件下，其他股东有优先购买权
- 视为同意：自接到通知之日起30日内未答复的，视为同意

### 股权转让协议核心条款

| 条款 | 审查要点 | 风险等级 |
|------|----------|----------|
| 股权状况 | 有无出资瑕疵、质押、冻结 | 高 |
| 对价及支付 | 定价依据、支付条件、时间 | 高 |
| 交割安排 | 工商变更、股东名册变更 | 中 |
| 陈述与保证 | 范围合理、有无兜底条款 | 中 |
| 竞业禁止 | 是否有竞业限制约定 | 中 |
| 违约责任 | 违约金数额、赔偿范围 | 中 |

## 股东协议审查

### 与章程的关系

| 效力层级 | 说明 |
|----------|------|
| 对内效力 | 股东协议在股东之间有效 |
| 对外效力 | 章程经登记具有公示效力 |
| 冲突处理 | 对外以章程为准，股东间违约方承担责任 |

### 常见股东协议条款

1. **投票权安排**：一致行动、委托投票
2. **否决权事项**：重大事项需一致同意
3. **回购/对赌条款**：业绩对赌、估值调整
4. **领售权/跟售权**：Tag-along / Drag-along
5. **反稀释条款**：价格保护机制
6. **优先认购权**：新股认购优先权
7. **信息权**：财务报表、重大事项知情

### 对赌条款效力

**最高法院指导意见**：
- 与目标公司对赌：原则上无效（违反资本维持原则），但目标公司有利润可分配时例外
- 与股东/实际控制人对赌：原则上有效

## 审查输出示例

```
风险等级：高
条款位置：公司章程第18条"对外担保"
现有表述："公司对外担保由董事会决定"

问题描述：
1. 未区分为股东/实际控制人担保与为非关联方担保
2. 未规定为股东担保须经股东会决议且关联股东回避
3. 未设定担保总额或单项担保限额
4. 可能导致担保决议程序瑕疵，影响担保效力

法律依据：《公司法》第15条第3款
"公司为公司股东或者实际控制人提供担保的，应当经股东会决议"

修改建议：
"第18条 对外担保
18.1 公司为他人提供担保，由董事会审议决定。但担保金额超过公司最近一期经审计净资产10%的，须经股东会审议决定。
18.2 公司为公司股东或实际控制人提供担保的，须经股东会决议。前款规定的股东或者受实际控制人支配的股东，不得参加前款规定事项的表决。该项表决由出席会议的其他股东所持表决权的过半数通过。
18.3 公司对外担保总额不得超过公司最近一期经审计净资产的50%。单笔担保金额超过公司最近一期经审计净资产的10%，须经股东会审议决定。"
```
