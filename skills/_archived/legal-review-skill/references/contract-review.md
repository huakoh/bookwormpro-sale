# 合同审查详细指南

## 适用法规

- 《民法典》第三编 合同（第463-988条）
- 最高人民法院关于适用《中华人民共和国民法典》合同编通则若干问题的解释

## 合同效力审查

### 无效情形（《民法典》第144、146、153、154条）

1. **主体资格缺陷**：无民事行为能力人实施的民事法律行为无效
2. **虚假意思表示**：行为人与相对人以虚假的意思表示实施的民事法律行为无效
3. **违反强制性规定**：违反法律、行政法规的强制性规定的民事法律行为无效
4. **违背公序良俗**：违背公序良俗的民事法律行为无效
5. **恶意串通**：行为人与相对人恶意串通，损害他人合法权益的民事法律行为无效

### 可撤销情形（《民法典》第147-151条）

1. **重大误解**：基于重大误解实施的民事法律行为
2. **欺诈**：一方以欺诈手段，使对方在违背真实意思的情况下实施的民事法律行为
3. **胁迫**：一方或者第三人以胁迫手段，使对方在违背真实意思的情况下实施的民事法律行为
4. **显失公平**：一方利用对方处于危困状态、缺乏判断能力等情形，致使民事法律行为成立时显失公平

## 核心条款审查清单

### 1. 合同主体条款

**审查要点**：
- 主体名称与统一社会信用代码准确性
- 法定代表人或授权代表身份
- 签约主体与履约主体一致性
- 担保人资格与能力

**常见问题**：
- 主体名称有误或已变更
- 签约代表无有效授权
- 以分支机构名义签约而总公司不承担责任

### 2. 标的条款

**审查要点**：
- 标的物描述明确、具体
- 数量、质量标准清晰
- 是否存在特许经营、资质要求

**常见问题**：
- 标的物描述模糊导致争议
- 质量标准约定不明

### 3. 价款与支付条款

**审查要点**：
- 价格构成明确（含税/不含税）
- 支付条件和时间节点
- 付款方式和账户信息
- 价格调整机制

**常见问题**：
- 发票类型未约定
- 付款条件过于苛刻或模糊

### 4. 违约责任条款

**审查要点**：
- 违约情形定义清晰
- 违约金数额合理（不超过实际损失的30%为宜）
- 损害赔偿范围明确
- 责任限制条款效力

**法律依据**：《民法典》第585条关于违约金的规定

### 5. 不可抗力条款

**审查要点**：
- 不可抗力定义是否符合法定标准
- 通知义务和证明要求
- 免责范围和后果处理

**法律依据**：《民法典》第180条、第590条

### 6. 争议解决条款

**审查要点**：
- 管辖法院或仲裁机构选择明确
- 仲裁条款有效性（仲裁机构必须明确）
- 管辖约定是否违反级别管辖和专属管辖

**常见问题**：
- "可以向法院起诉或向仲裁机构申请仲裁"——此类条款无效
- 约定的仲裁机构不存在

## 格式条款特别审查

**法律依据**：《民法典》第496-498条

**审查要点**：
1. 是否履行提示说明义务
2. 是否存在免除己方责任、加重对方责任、排除对方主要权利的条款
3. 格式条款与非格式条款不一致时，以非格式条款为准

**高风险格式条款示例**：
- "乙方放弃一切索赔权利"
- "甲方对任何损失不承担责任"
- "合同解释权归甲方所有"

## 审查输出示例

**示例：价款条款问题**

```
风险等级：中
条款位置：第5.1条
现有表述："乙方应在收到货物后支付款项"
问题描述：付款期限不明确，可能导致付款义务争议
法律依据：《民法典》第511条第4项，履行期限不明确的处理
修改建议：修改为"乙方应在收到货物并验收合格后15个工作日内支付款项"
```
