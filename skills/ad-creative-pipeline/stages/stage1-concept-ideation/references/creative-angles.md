# 广告创意角度库

> Stage 1 参考文件。每个角度含触发场景、标题公式、视觉方向、Prompt 模板。

---

## 1. Pain Point (痛点)

**触发场景:** 功能性产品、B2B、问题解决型产品
**心理机制:** 损失厌恶 — 人们对损失的恐惧远大于对收益的渴望

### 标题公式
```
[问题成本] + [解决方案暗示]
例: "每一度温差，都是百万损失" / "90% 的冷链事故，发生在你看不到的地方"
```

### 视觉方向
- 问题可视化: 展示问题发生的瞬间或后果
- 对比手法: "有 vs 没有"的视觉对比
- 负面情绪色彩: 冷色调、暗光、紧张感

### Prompt 模板
```
展示 [行业/场景] 中 [具体问题] 的视觉呈现。
色调: [冷色调/暗色]
情绪: [紧张/焦虑/损失感]
关键元素: [问题的视觉符号]
避免: 解决方案元素（那是 Benefit 角度的事）
```

---

## 2. Benefit (利益)

**触发场景:** 所有场景，最通用的角度
**心理机制:** 趋利 — 展示使用产品后的正面结果

### 标题公式
```
[用户] + [动词] + [正面结果]
例: "你的药品，我们守护每一公里" / "让合规不再加班"
```

### 视觉方向
- 结果展示: 使用产品后的理想状态
- 正面情绪: 暖色调、明亮、满足感
- 人物受益: 真人使用场景（B2C）或场景改善（B2B）

### Prompt 模板
```
展示 [用户/场景] 享受 [产品带来的好处]。
色调: [暖色调/明亮]
情绪: [满足/放心/高效]
关键元素: [产品融入场景，自然不突兀]
```

## 3. Social Proof (社交证明)

**触发场景:** B2B、高客单价、信任门槛高的产品
**心理机制:** 从众 — "别人都在用，应该没问题"

### 标题公式
```
[数字] + [权威群体] + [行为]
例: "500+ 药企的共同选择" / "行业 Top 10 中 7 家的冷链伙伴"
```

### 视觉方向
- 数据可视化: 数字、图表、客户 Logo 墙
- 群体场景: 多人使用/会议/签约
- 信任符号: 认证徽章、奖杯、推荐信

### Prompt 模板
```
展示 [数量] 个 [客户类型] 的 [使用场景/Logo阵列]，
带有 [认证/奖项] 的视觉元素。
色调: [专业/信任色 — 蓝、灰、白]
风格: 企业专业风，干净简洁
```

## 4. Authority (权威)

**触发场景:** B2B、医疗、金融、法律
**心理机制:** 服从权威 — "专家/认证机构说好，肯定没问题"

### 标题公式
```
[权威标准/认证] + [产品能力]
例: "GSP 合规，从第一公里到最后一公里" / "通过 ISO 13485 认证的智能温控"
```

### 视觉方向
- 认证/资质展示: GSP、ISO、FDA 徽章
- 专家形象: 白大褂、实验室、专业设备
- 精密感: 数据仪表盘、技术图示

### Prompt 模板
```
展示 [认证/标准名称] 的视觉符号，
结合 [行业专业场景]。
色调: [专业蓝/白]
元素: 认证徽章、精密仪器、数据界面
```

## 5. Curiosity (好奇)

**触发场景:** 新品类、创新产品、需要教育市场的产品
**心理机制:** 信息缺口 — 人类天生想填补知识空白

### 标题公式
```
[反常识陈述] 或 [悬念问题]
例: "你的冷链，可能一直在'裸奔'" / "为什么药企巨头都在换温控系统？"
```

### 视觉方向
- 悬念画面: 局部特写/模糊暗示/未完成构图
- 反常识: 打破预期的视觉
- 互动暗示: "点开看答案" 的视觉引导

### Prompt 模板
```
一个引发好奇的视觉画面: [局部特写/悬念构图]。
构图: 刻意不完整/模糊化处理 [某元素]，
引导观众想看更多。
色调: [对比强烈/有视觉张力]
```

## 6. Competitor (竞品对比)

**触发场景:** 成熟市场、有明显差异化优势的产品
**心理机制:** 对比效应 — 放在一起比，好坏立现

### 标题公式
```
[对比维度] + [你的优势]
例: "别人还在看温度，我们已经在预测风险" / "普通 vs AVITS: 差的不只是 ±0.1°C"
```

### 视觉方向
- 对比分屏: 左右/上下分割，一侧差一侧好
- 数据对比: 柱状图、进度条
- 警示色 vs 安心色

### Prompt 模板
```
分屏构图的广告画面: 左侧展示 [竞品/旧方案的不足]，
右侧展示 [产品的优势]。
左侧色调 [暗淡/问题色]，右侧色调 [明亮/品牌色]。
视觉对比强烈，一目了然。
```

---

## 7. Urgency (紧迫感)

**触发场景:** 促销活动、限时优惠
**心理机制:** 稀缺 — 越少越想拥有，越急越快行动

### 标题公式
```
[限时/限量] + [行动号召]
例: "首月免费，仅剩 3 天" / "前 100 名享专属折扣"
```

### 视觉方向
- 时间元素: 倒计时、日历、钟表
- 抢购感: 动态模糊、红色强调
- 限量符号: "仅剩 X"、"限量" 标签

---

## 8. Story (故事)

**触发场景:** 品牌建设、情感连接
**心理机制:** 叙事传输 — 故事让人放下防御，产生共情

### 标题公式
```
[人物] + [转变] + [产品角色]
例: "从每天提心吊胆到安心睡个好觉 — 一个药企物流经理的 30 天"
```

### 视觉方向
- 人物故事: 真人出镜、前后对比
- 电影感: 叙事画面、暖色调
- 时间线: 过程展示

### Prompt 模板
```
叙事风格的广告画面，展示 [人物] 的 [转变时刻]。
电影感构图，[色调描述]。
人物表情 [从X到Y的变化]。
自然光、真实质感。
```
