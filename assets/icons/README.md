# BookwormPRO 图标集

此目录包含基于 quantum-logo-suite 设计语言通过 Seedream 5.0 生成的图标。

## 设计语言

- 主色：金色 (#C8A050) + 青色 (#20E0C0) + 深蓝 (#4080FF)
- 背景：深邃太空黑 (#06060C)
- 主题：量子旋转书虫

## 生成方式

```
python scripts/generate_icons.py
```

使用 Seedream 5.0 (doubao-seedream-5-0-260128) 生图，
doubao-seed-1-6-vision-250815 自动四维评审。
