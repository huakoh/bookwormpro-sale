一、这是什么

BookwormPRO 是一套为 AI 编程助手设计的技能生态系统。它不改变你的 AI 工具——而是让你的 AI 工具在遇到特定任务时，自动加载对应的专业领域知识和工作流程。

核心理念：路由引擎推荐了专家，专家就必须上场。每次对话中，AI 自动识别你的意图，匹配最合适的技能，加载完整专业指导——你不需要记住 305 个技能的名字。

BookwormPRO 是配置和技能包，不是独立应用。它需要配合支持 Skills 的 AI 编程助手使用（如 Claude Code、Bookworm Agent 等）。

二、前置条件

项目要求
Python3.10+
Git任意版本
AI 助手支持 Skills 的编程助手（Claude Code / Bookworm Agent 等）
API Key至少一个 LLM 服务商的 Key（推荐 DeepSeek）

三、安装步骤

1 克隆仓库 
git clone https://github.com/huakoh/bookwormpro-sale.git /tmp/bookwormpro-sale
2 运行安装脚本 
cd /tmp/bookwormpro-sale
python scripts/setup.py

脚本会引导你填入 API Key。选"快速模式"跳过交互：

python scripts/setup.py --quick
3 验证安装

重启 AI 助手后输入：

bookworm自检

如果看到 6 环节体检报告，安装成功 ✅

手动安装（脚本不可用时）

mkdir -p ~/.bookwormpro

# 复制技能（核心）
cp -r skills/ ~/.bookwormpro/skills/

# 复制灵魂文件
cp soul/SOUL.md ~/.bookwormpro/
cp soul/SOUL.md ~/.claude/

# 复制配置模板
cp config/.env.template ~/.bookwormpro/.env
cp config/config.yaml ~/.bookwormpro/config.yaml

# 编辑 .env 填入你的 API Key
nano ~/.bookwormpro/.env

四、配置 API Key

打开 ~/.bookwormpro/.env，最少需要配置一个 Provider：

推荐配置（DeepSeek，性价比最高）

DEEPSEEK_API_KEY=sk-your-deepseek-key-here
DEEPSEEK_BASE_URL=https://api.deepseek.com/v1

完整配置示例

# ── 必选（至少配一个）──
DEEPSEEK_API_KEY=sk-your-key-here
DEEPSEEK_BASE_URL=https://api.deepseek.com/v1

# ── 可选：解锁更多能力 ──
DASHSCOPE_API_KEY=sk-your-key          # 通义千问 + 图片生成 + 视觉
DASHSCOPE_BASE_URL=https://dashscope.aliyuncs.com/compatible-mode/v1
AUXILIARY_VISION_MODEL=qwen-vl-max     # 视觉识别模型

GOOGLE_API_KEY=your-key                # Gemini

# 豆包视觉（国内方案）
DOUBAO_API_KEY=ark-your-key
DOUBAO_BASE_URL=https://ark.cn-beijing.volces.com/api/v3

切换主 Provider

编辑 ~/.bookwormpro/config.yaml：

model:
  provider: deepseek       # 改为 openai / anthropic
  default: deepseek-v4-pro # 改为你用的模型
获取 DeepSeek API Key：访问 platform.deepseek.com 注册并充值（支持支付宝）。

七、人格系统

BookwormPRO 默认激活"见山大叔"人格——敲过二十年代码的老派工程师：

  • 中文回复，技术名词保留英文
  • 先分析后动手，给方案列编号
  • 改代码必附 diff——改了什么、为什么、影响什么
  • 不留 TODO，不交半成品，质量过四关才出手
  • 专业但不端着，务实但有温度

编辑 ~/.bookwormpro/config.yaml 可自定义人格或禁用它（设置 display.personality 为空）。

创建自定义人格：

# 使用 bookwormpro-custom-personality 技能
# AI 会引导你一步步创建专属人格

八、系统自检

BookwormPRO 内置了 6 环节自检管线，随时诊断系统健康：

命令执行内容
bookworm自检全量 6 环节体检
bookworm自检 修复全量体检 + 自动修复已知问题
bookworm自检 只看cron仅检查定时任务
bookworm自检 只看env仅审查 .env 配置
bookworm自检 只看日志仅分析日志噪音
bookworm自检 只看凭证仅查凭证冲突
bookworm自检 只看连通仅测 Provider 连通性
建议每周跑一次 bookworm自检，及时发现配置漂移或 API Key 过期。

十、常见问题

Q: 提示 "skill not found"

确认 ~/.bookwormpro/skills/ 目录存在且有 SKILL.md 文件。运行 python scripts/setup.py 重新安装。

Q: API 调用返回 401/403

检查 ~/.bookwormpro/.env 中 API Key 是否正确，确认账户余额充足。API Key 不要加引号。

Q: 怎么知道 AI 用了哪个技能

对话中会出现 [BWR:xxx] 路由标识，标明置信度和匹配的技能。或者直接用 /技能名 强制指定。

Q: 想增加新技能怎么办

把 SKILL.md 文件放到 ~/.bookwormpro/skills/你的分类/ 目录下即可。AI 会自动发现新技能。

Q: 支持哪些 AI 工具

支持所有兼容 Skills 协议的 AI 编程助手，包括 Claude Code、Cursor、Windsurf、Bookworm Agent 等。

Q: 安装后要重启吗

需要重启你的 AI 助手，让它重新扫描 skills 目录。

Q: API Key 安全吗

Key 存储在本地 ~/.bookwormpro/.env，不会被上传到任何服务器。BookwormPRO 遵守退化安全网原则：绝不暴露凭证明文于代码/日志/响应中。

不要将 .env 文件提交到 Git 仓库。.gitignore 已默认排除它。
