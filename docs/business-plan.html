Section 01

产品定位

BookwormPRO 是开发者本地 AI 操作系统 -- CLI 原生，多模型路由，73 个专家技能即装即用。对标 Claude Code / Gemini CLI 赛道。

关键差异化

DIFF 01
CLI 原生可编程性
Skill 路由 + Hook 可编程性 + 多 MCP 协作，终端原生工作流。
DIFF 02
73 技能全栈覆盖
开发、AI/ML 为核心，商务/法务/财务为 LLM prompt 辅助，即装即用。
DIFF 03
与 Web UI 互补
与 Dify/Coze 等可视化平台互补而非竞争，走 CLI 原生差异化路线。
诚实边界 -- 不可对外声称
  • X 不说"数据零上传" — 使用云 API 时数据经第三方处理
  • X 不说"替代 15 万/月团队" — 过度承诺，损害品牌信任
  • X 不说"国内没有成熟产品" — Dify 已有 132K+ Star

收入模式调整 (关键修正)

重要调整

原方案：中转站差价 60% + 订阅 30% + Skills Hub 10%

修正后：纯 License 订阅为主 (毛利 85%) + Skills Hub (长期)

原因：中转站需增值电信许可证 (法律硬性障碍) + 中转毛利仅 30-50%，不值得冒合规风险。

Section 05

安装体验设计

目标：一键下载，双击安装，马上使用。MVP 开发时间 6-7 个工作日（复用千牛项目经验）。

三平台方案

PLATFORM 01
Windows
PyInstaller onedir + NSIS 安装器，约 80MB，内嵌 Python 3.11。
PLATFORM 02
macOS
PyInstaller .app + create-dmg 打包 .dmg，符合 macOS 用户习惯。
PLATFORM 03
Linux
AppImage 单文件零依赖 + deb 包，覆盖主流发行版。

首次运行 Setup Wizard

1
选择 Provider
DeepSeek / Kimi / OpenRouter / OpenAI，下拉选择。
2
粘贴 API Key
即时验证 + "获取 Key" 跳转链接，失败时给出精准错误提示。
3
选择模型
Key 验证通过后自动拉取可用模型列表，不再手动填写。
4
一键完成
73 技能 + 9 MCP + 50 钩子自动就绪，进入主界面。
