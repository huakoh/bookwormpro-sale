❓ 常见问题

双击 install.bat 没反应？

右键 → 以管理员身份运行。或者打开终端，cd 到文件夹，输入 python scripts/setup_wizard.py

提示"未检测到 Python"？

去 python.org 下载安装。安装时一定要勾选 "Add Python to PATH"，然后重新双击 install.bat。

哪里获取 API Key？

推荐 DeepSeek 开放平台 → 注册 → API Keys → 创建。支持支付宝充值，1 元起。

怎么知道装成功了？

双击 check.bat 看诊断报告。或者打开 AI 助手输入 bookworm自检。
