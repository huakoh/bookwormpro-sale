准备安装配置激活使用

0先准备两样东西 1 分钟

电脑上装过旧版 Bookworm？先清理

有 bookworm 命令就运行 bookworm uninstall。
太早的版本没这个命令？手动删：

Remove-Item -Recurse -Force "$env:LOCALAPPDATA\bookworm","$HOME\.bookwormpro","$HOME\.hermes" -ErrorAction SilentlyContinue

删完重开终端再继续。

第一样：Git

检查是否已装：打开终端，输入 git --version。有输出就行，没有就装：

winget install Git.Git

装好后关闭终端再打开，输入 git --version 能看到版本号即可。

第二样：AI API Key

BookwormPRO 需要调用 AI 模型。最简单的方式：

打开 platform.deepseek.com → 注册 → 支付宝充 1 元 → 复制 API Key → 记下来待会用。

你手里应该有一串 sk- 开头的字符串。

没有国际信用卡也能用吗？

能。DeepSeek 支持支付宝充值。或者用 源力AI 中转站，也支持支付宝，一个 Key 还能调用 GPT-4o、Claude 等多家模型。

准备好了，下一步：安装 ↓

1一键安装 3 分钟

打开终端 (按 Win+X → 选 "终端")，粘贴这一行：

irm https://raw.githubusercontent.com/huakoh/bookwormpro-sale/master/scripts/install.ps1 | iex

等它跑完（自动安装 Python、Node.js 等依赖），然后 关闭终端，重新打开。

打开终端，粘贴这一行：

curl -fsSL https://raw.githubusercontent.com/huakoh/bookwormpro-sale/master/scripts/install.sh | bash

跑完后执行 source ~/.bashrc

git clone https://github.com/huakoh/bookwormpro-sale.git
cd BookwormPRO && docker compose up -d --build

验证：在新终端里输入

bookworm version

应该看到 BookwormPRO v7.0.0 或更高版本号。看到了就说明装好了。

输入 bookworm 提示 "不是命令" 怎么办？

你没有重新打开终端。关闭当前终端窗口，重新打开一个新的再试。
还不行？手动加 PATH：$env:PATH += ";$env:LOCALAPPDATA\bookworm\bookwormpro\venv\Scripts"

安装完成，下一步：配置 AI ↓

2配置 AI 模型 30 秒

安装过程中向导会自动弹出。如果跳过了，运行：

bookworm setup --quick

向导问你选哪个 AI 时，选 DeepSeek，然后粘贴刚才复制的 API Key。

向导结束后会显示 "Setup complete" 和已配置的模型名称。

我想用其他 AI 模型怎么办？

运行 bookworm setup model 进入完整配置。支持 28 个 Provider：Anthropic、OpenAI、Gemini、Kimi、通义等。
如果用 源力AI 中转站，选 "Custom endpoint"，Base URL 填 https://bww.letcareme.com/v1。

配置完成，下一步：激活 ↓

3激活 10 秒

先免费试用 7 天，满意了再购买：

bookworm trial

看到绿色的 "Trial activated!" 框就成功了。显示到期日期和 tier: trial。

无需注册、无需信用卡、无需联网验证。每台电脑可试用一次。

提示 "Trial already used" 怎么办？

这台电脑已经试用过了。联系客服购买正式 License：查看定价。

已购买？激活正式 License 
# 1. 先拿到机器指纹
bookworm license hwid
# 输出一串 64 位字符，复制它

# 2. 把指纹发给客服，收到 .license 文件后：
bookworm activate 你的文件.license
# 看到 "Activation Successful" 即可

# 3. 确认一下
bookworm license status
激活完成，开始使用! ↓

Go开始对话

在终端输入：

bookworm

看到 BookwormPRO 的欢迎横幅和 > 提示符，就可以开始对话了。
直接用中文打字，AI 会自动匹配最合适的专家技能。

试试说：

"帮我审查这个项目的安全性"

自动调用安全审查技能组合

"帮我写一个 Python 爬虫"

自动匹配开发专家技能

"分析这份 sales.csv"

自动启动数据分析 + 出图表

"/help"

查看全部命令

进阶：其他启动方式 
bookworm --tui         # 图形化界面 (鼠标操作/模态窗口)
bookworm --continue    # 继续上次没聊完的对话
进阶：安装后体检 
bookworm canary            # 快速检查 (不联网)
bookworm canary --live      # 检查 + 测试 API 连通
bookworm doctor             # 完整诊断报告

遇到问题？运行 bookworm dump 生成调试信息，发给技术支持。

进阶：常用命令速查

配置管理

bookworm setup          # 完整向导
bookworm setup model    # 只改模型
bookworm setup gateway  # 消息平台

License 管理

bookworm license status     # 状态
bookworm license hwid       # 指纹
bookworm license deactivate # 移除

Gateway

bookworm gateway setup   # 配置
bookworm gateway start   # 启动
bookworm gateway status  # 状态

维护

bookworm update      # 更新版本
bookworm doctor      # 诊断
bookworm uninstall   # 卸载
