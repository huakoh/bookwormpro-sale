6 故障排查

bookworm: command not found

关闭并重新打开终端。如果仍然不行，手动添加 PATH：
Windows: $env:PATH += ";$env:LOCALAPPDATA\bookworm\bookwormpro\venv\Scripts"
Linux: export PATH="$HOME/.bookwormpro/bookwormpro/venv/bin:$PATH"

AI 回复为空或报错

重新运行 bookworm setup model 检查 Provider 和 API Key 配置。
运行 bookworm doctor 查看详细诊断。

Gateway 启动后平台无响应

运行 bookworm gateway status 检查各平台连接状态。
重新运行 bookworm gateway setup 更新 Token。

已有旧版本，想重新安装

先运行 bookworm uninstall 卸载旧版，然后重新按安装指南操作。
如果旧版本太早没有 uninstall 命令，手动清理：
Remove-Item -Recurse -Force "$env:LOCALAPPDATA\bookworm","$HOME\.bookwormpro","$HOME\.hermes" -ErrorAction SilentlyContinue

Python 安装失败 / 下载超时

安装脚本会自动通过 uv 和 winget 两种方式安装 Python。如果都失败 (常见于网络受限环境)，手动安装后重试：
winget install Python.Python.3.12
安装完成后关闭并重新打开终端，再次运行安装脚本即可。

Python/Node 版本冲突

BookwormPRO 使用独立虚拟环境，不影响系统 Python。
如遇冲突，删除 %LOCALAPPDATA%\bookworm\bookwormpro\venv 目录后重新安装。

常见问题

安装与使用

需要什么前置条件?

只需要 Git 和一个 AI API Key。推荐 DeepSeek (platform.deepseek.com，支付宝充值 1 元起)。安装脚本自动处理 Python、Node.js 等其他依赖。

支持哪些 AI 助手?

所有兼容 Skills 协议的 AI 编程助手：Claude Code、Cursor、Windsurf 等。同时内置独立 CLI / TUI 客户端和 Gateway 多平台服务。

和直接问 AI 有什么区别?

AI 本身是通用模型。BookwormPRO 给它加载了 135+ 个专家级的提示词和工作流程。同样的 AI，加载技能后输出的代码质量、安全审查深度、设计水平显著提升。

怎么更新?

运行 bookworm update 即可。更新不丢失你的 API Key、License 和个人配置。

License 与激活

HWID mismatch 怎么办?

License 绑定的机器与当前机器不同。运行 bookworm license hwid 获取新 HWID 联系客服重新绑定。更换电脑、重装系统、更换网卡都可能导致 HWID 变化。

License expired 怎么办?

联系客服续费 (续费 ¥210/年，7 折)。收到新 .license 文件后重新 bookworm activate 即可。

可以在多台机器上使用吗?

个人版绑定单台机器 (HWID)。多台机器使用请联系客服获取团队/企业多机授权方案。

Skills 列表为空?

可能 License 未激活或已过期。运行 bookworm license status 检查。加密技能需要有效 License 解锁。

安全与支付

我的 API Key 安全吗?

Key 只存储在本地 ~/.bookwormpro/.env 文件，不上传任何服务器。技能文件采用 AES-256-GCM 加密，License 采用 Ed25519 数字签名，防篡改防拷贝。

可以开发票吗?

可以。提供电子普通发票。联系客服告知开票信息即可。
