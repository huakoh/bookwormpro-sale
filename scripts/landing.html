6 故障排查

bookworm: command not found

关闭并重新打开终端。如果仍然不行，手动添加 PATH：
Windows: $env:PATH += ";$env:LOCALAPPDATA\bookworm\bookwormpro\venv\Scripts"
Linux: export PATH="$HOME/.bookwormpro/bookwormpro/venv/bin:$PATH"

AI 回复为空或报错

重新运行 bookworm setup model 检查 Provider 和 API Key 配置。
运行 bookworm doctor 查看详细诊断。

Gateway 启动后平台无响应

运行 bookworm gateway status 检查各平台连接状态。
重新运行 bookworm gateway setup 更新 Token。

Python/Node 版本冲突

BookwormPRO 使用独立虚拟环境，不影响系统 Python。
如遇冲突，删除 %LOCALAPPDATA%\bookworm\bookwormpro\venv 目录后重新安装。

激活 License

购买后收到 .license 文件，3 步完成激活

1 获取机器指纹 (HWID)

首次激活需要采集机器硬件指纹，用于绑定 License。

bookworm license hwid
# 输出: Machine HWID: 24afab...4ac71c
# 将此 HWID 发送给客服，客服据此生成专属 License

2 激活 License 文件

收到客服发来的 .license 文件后，一行命令激活。

bookworm activate /path/to/your.license
# ┌─────────── Activation Successful ───────────┐
# │ License activated                           │
# │   Licensee:  Your Company                   │
# │   Tier:      pro                            │
# │   Expires:   2027-06-01                     │
# └─────────────────────────────────────────────┘

3 验证激活状态

bookworm license status
# ┌────────────── License Status ──────────────┐
# │   Status:    VALID                         │
# │   Licensee:  Your Company                  │
# │   Tier:      pro                           │
# │   Expires:   2027-06-01                    │
# └────────────────────────────────────────────┘

License 管理命令

查看状态

bookworm license status

查看当前 License 有效性、到期时间、绑定信息。

查看机器指纹

bookworm license hwid

显示当前机器 HWID。更换机器后需重新绑定。

移除 License

bookworm license deactivate

从本机移除已安装的 License 文件。

环境变量模式

export BOOKWORMPRO_LICENSE_KEY="key"

适用于 CI/CD 或容器环境，免文件激活。

常见激活问题

HWID mismatch 怎么办?

说明 License 绑定的机器与当前机器不同。运行 bookworm license hwid 获取新 HWID，联系客服重新绑定。更换电脑、重装系统、更换网卡都可能导致 HWID 变化。

License expired 怎么办?

联系客服续费。续费价格 ¥210/年 (7 折)。续费后会收到新的 .license 文件，重新 bookworm activate 即可。

Invalid signature 怎么办?

License 文件被修改或损坏。请重新从客服获取原始 .license 文件。请勿手动编辑该文件。

Skills 列表为空?

可能是 License 未激活或已过期。运行 bookworm license status 检查状态。加密技能需要有效 License 才能解锁。

可以在多台机器上使用吗?

每个 License 绑定一台机器 (HWID)。如需多台机器使用，请联系客服购买额外席位或企业版。

和直接问 AI 有什么区别?
