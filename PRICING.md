## 免费版

适合个人开发者、学生、技术爱好者。

**包含：**
- 100 个核心开发技能（developer-expert, debugger-expert, security-expert...）
- 默认人格系统
- 手动安装（ZIP 解压 → 复制文件）
- 社区支持（GitHub Issues）

**获取方式：** 从 GitHub Releases 下载 `bookwormpro-free-vX.X.X.zip`

**包含：**

**获取方式：** [购买 Pro 版](https://example.com/buy-pro) → 获取 License Key → 下载完整包

## 常见问题

**Q: 免费版和 Pro 版技能质量一样吗？**
A: 一样。免费版的 100 个核心技能与 Pro 版同源，质量无差别。Pro 版多出的 205 个技能覆盖更多垂直领域。

**Q: Pro 版是买断还是订阅？**
A: 年付订阅。包含一年更新和技术支持。续费享 7 折。

**Q: 可以先用免费版再升级吗？**
A: 可以。升级后直接覆盖安装即可，API Key 等个人配置不受影响。

**Q: 企业版可以私有部署吗？**
A: 可以。企业版提供离线安装包，无需外网。支持内网 Git 仓库同步更新。

**Q: 支持哪些 AI 助手？**
A: 所有兼容 Skills 协议的 AI 编程助手，包括 Claude Code、Cursor、Windsurf、Bookworm Agent 等。
